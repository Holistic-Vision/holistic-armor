# Script audio — Circuit full body doux (30 min)
Voix: Neutre (coach)
Règles:
- Ton calme, direct, non médical
- Pauses indiquées par [pause 2s]
- Ne jamais mentionner diagnostic, traitement, “cause hormonale”
## Texte
[pause 2s] On démarre. Intensité confortable.
[pause 2s] Si une douleur aiguë apparaît : stop.
[pause 2s] Concentre-toi sur la qualité du mouvement.
[pause 2s] Respire régulièrement.
[pause 2s] Objectif : bouger mieux, pas plus.
[pause 2s] Fin : note ton énergie et ton ressenti.
