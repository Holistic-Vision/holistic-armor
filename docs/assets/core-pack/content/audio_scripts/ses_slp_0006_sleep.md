# Script audio — Relâchement corps (7 min)
Voix: Neutre (sommeil)
Règles:
- Ton calme, direct, non médical
- Pauses indiquées par [pause 2s]
- Ne jamais mentionner diagnostic, traitement, “cause hormonale”
## Texte
[pause 2s] Ce moment est pour récupérer.
[pause 2s] Lumière douce. Écran éteint.
[pause 2s] Inspire tranquillement…
[pause 2s] Expire plus longuement…
[pause 2s] Relâche le front… la nuque… les épaules…
[pause 2s] Les pensées passent. Tu reviens au souffle.
[pause 2s] Tu n’as rien à résoudre maintenant.
